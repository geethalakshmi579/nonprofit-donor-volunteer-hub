# Nonprofit Donor & Volunteer Management Tool

**AI-Powered Web App for Small Nonprofits**

**Live Demo**: [https://geethalakshmi579-nonprofit-donor-volunteer-hub-app-8vy2xd.streamlit.app/](https://geethalakshmi579-nonprofit-donor-volunteer-hub-app-8vy2xd.streamlit.app/)

## 🎯 Project Overview
A complete end-to-end Streamlit application that helps small nonprofits manage donors and volunteers more efficiently. It automates data tracking, generates insights, creates personalized communications, and produces professional reports.

## ✨ Key Features
- **Interactive Dashboard** with KPIs and visualizations
- **Data Explorer** — powerful search, filters, and pivoting
- **AI Insights** — smart analysis and actionable recommendations (Claude-powered)
- **Personalized Email Generator** — thank-you and follow-up emails
- **Reports** — Export to CSV and PDF
- Sample data + CSV/Excel file upload support

## 🛠️ Tech Stack
- **Framework**: Streamlit
- **Data**: Pandas
- **Charts**: Plotly
- **PDF Generation**: FPDF
- **AI**: Claude (mock implementation for public demo)

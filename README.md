## 🎯 Project Overview
A complete end-to-end web application designed to help small nonprofits efficiently manage donors, volunteers, generate insights, and automate communications.

**Live Demo**: [https://geethalakshmi579-nonprofit-donor-volunteer-hub-app-8vy2xd.streamlit.app/](https://geethalakshmi579-nonprofit-donor-volunteer-hub-app-8vy2xd.streamlit.app/)

## ✨ Key Features
- **Interactive Dashboard** with KPIs and visualizations (Plotly)
- **Data Explorer** — advanced search, filtering, and pivoting
- **AI Insights** — intelligent analysis and actionable recommendations
- **Personalized Email Generator** — bulk thank-you & follow-up emails
- **Professional Reports** — Export to CSV and PDF
- Sample data generator + support for CSV & Excel uploads

## 🛠️ Tech Stack
- **Framework**: Streamlit
- **Data Processing**: Pandas
- **Visualization**: Plotly
- **Reporting**: FPDF
- **AI**: Claude (mock responses for public demo)

## How I Built It
I developed this tool as part of my preparation for the **Claude Corps Fellowship**. I used Claude extensively for code generation, feature ideation, UI improvements, and debugging. 

This project helped me:
- Understand real-world nonprofit challenges (donor retention, volunteer management, reporting burden)
- Build a production-ready full-stack application
- Demonstrate practical application of AI tools for social impact
